ubuntu 20.04 安装搜狗输入法

在ubuntu 20.04中，搜狗输入法依赖的qt4被移除了（目前的官方版本已经解决了依赖问题），所以无法使用，可以使用麒麟社区的定制版本。

不过搜狗输入法极易造成pycharm等软件卡死，目前我在用的是rime输入法（其输入法系统为ibus而不是fctix，兼容性比较好。没有搜狗输入法智能，但是比系统自带输入法智能）。

搜狗输入法定制版

1.1 安装

# 添加软件源
curl -sL 'https://keyserver.ubuntu.com/pks/lookup?&op=get&search=0x73BC8FBCF5DE40C6ADFCFFFA9C949F2093F565FF' | sudo apt-key add
sudo apt-add-repository 'deb http://archive.ubuntukylin.com/ukui focal main'
sudo apt update

# 安装搜狗输入法
sudo apt-get install sogouimebs

命令较长，可访问网址 https://download.likole.com/shurufa.html 来复制内容。

1.2 设置默认输入法

进入设置，选择“区域与语言”，进入“管理已安装的语言”，把键盘输入法系统改为“fcitx”，此外需要在fctix configuration中添加搜狗输入法（可能已经添加了），重启电脑即可使用。

1.3 禁用ibus（可选）

fcitx有时候会与ibus发生冲突造成卡死，可以禁用掉ibus。

# 禁用ibus
sudo dpkg-divert --package im-config --rename /usr/bin/ibus-daemon

# 附：启用方法
sudo dpkg-divert --package im-config --rename --remove /usr/bin/ibus-daemon
